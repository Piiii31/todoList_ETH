# Portfolio Template

Copy this `.portfolio-template` folder to your GitHub repositories and rename it to `.portfolio`.

## Quick Setup:

1. Copy this entire folder to your repo
2. Rename `.portfolio-template` to `.portfolio`
3. Edit each file with your project information
4. Add images to the `images/` folder
5. Commit and push to GitHub

## Files to Edit:

- **description.md** - Your project description (supports markdown)
- **features.md** - List of features (one per line)
- **stack.json** - Technologies you used
- **related.json** - Slugs of related projects
- **meta.json** - Status, progress, fun fact, category
- **images/** - Add thumbnail.png and screenshots

## Image Guidelines:

- **thumbnail.png** - Main project image (1200x675px recommended, 16:9 ratio)
- **screenshot-1.png** through **screenshot-6.png** - Optional screenshots
- Use PNG or JPG format
- Keep file sizes reasonable (< 500KB each)

Your portfolio website will automatically fetch and display this data!
